# Grafcet Design

此軟體根據MIAT方法論中的離散事件建模Grafcet設計

Grafcet Design 的功能有:
- Grafcet圖形編輯
- Grafcet檔案儲存
- Grafcet檔案載入
- C code 自動生成
- VHDL code 自動生成

可支援Sub-Grafcet圖形編輯與程式碼生成
